# 🎬 MovieTracking

A Python-based application for managing and displaying movie content, showtimes, and seat selections using Pygame.

---

## 📦 Features

- Movie and showtime selection interface
- Seat selection system with dynamic layout
- Interactive UI powered by Pygame
- Modular, object-oriented design
- Real-time event handling and state updates

---

## 🛠️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/MovieTracking.git
cd MovieTracking
```

### 2. Create an environment
```
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

pip install -r requirements.txt
```

### 3. Run the program (Must run it from the root folder)
```python main.py```

## 📝 Notes
Requires Python 3.11.4+

Built with Pygame for educational/demo purposes

Compatible with Windows, macOS, and Linux

Tested on an M2 Macbook Air
